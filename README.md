# 就職活動支援アプリ 要件定義書

## 1. システム概要

本システムは、大学生の就職活動における企業管理や選考スケジュール管理を支援することを目的とし>

企業ごとのエントリー情報や選考状況、提出期限などを一元管理し、一覧表示やカレンダー表示、通知>

---

## 2. システムの目的

* 就職活動に関する情報を一元管理する。
* ES提出や面接日程などの重要な期限を管理する。
* 通知機能により提出忘れや日程の見落としを防止する。
* 学生の就職活動の負担を軽減する。

---

## 3. 利用対象者

* 就職活動を行う大学生
* インターンシップや本選考を受ける学生

---

## 4. 機能要件

### 4.1 企業情報管理機能

企業ごとに選考状況や提出期限を登録・編集・削除できる。

#### 登録項目

| 項目名    | 内容            |
| ------ | ------------- |
| 企業名     | 応募企業名              |
| ID      | 企業サイトのログインID       |
| パスワード   | 企業サイトのログインパスワード    |
| URL     | 企業のマイページURL        |
| メールアドレス | 企業との連絡に使用するメールアドレス |
| 備考      | 自由記述欄              |
| ES提出日  | エントリーシートの提出期限 |
| ES提出状況 | 未提出・提出済       |
| 顔写真提出日 | 顔写真の提出期限      |
| SPI提出日 | SPI受験期限       |
| インターンシップ予定日 | インターンシップの日程       |
| 第1次面接日 | 一次面接の日程       |
| 第2次面接日 | 二次面接の日程       |
| 第3次面接日 | 三次面接の日程       |
| 備考     | 自由記述欄         |

---

### 4.2 一覧表示機能

登録されたタスクやイベントを一覧形式で表示する。

#### 機能内容

* 期限が近い順に表示する。
* 未完了のタスクを優先して表示する。
* ES提出や面接予定を一覧で確認できる。
* 完了済みタスクの表示切替を行う。

---

### 4.3 カレンダー表示機能

登録された予定をカレンダー形式で表示する。

#### 機能内容

* ES提出日を表示する。
* SPI受験日を表示する。
* 面接日程を表示する。
* 日付を選択すると詳細情報を確認できる。

---

### 4.4 通知機能

登録された予定の期限が近づいた際に通知を行う。

#### 通知方法

* アプリ内通知
* メール通知

#### 通知設定

* 通知機能のON/OFF切替
* 通知タイミング設定

  * 1日前
  * 3日前
  * 1週間前

---

## 5. 画面要件

### 5.1 一覧画面（ToDoリスト画面）



#### 表示内容

* 企業名
* タスク内容
* 期限
* 選考状況

#### 主な機能

* 期限順表示
* 完了・未完了切替
* タスク詳細表示

---

### 5.2 カレンダー画面

#### 表示内容

* ES提出日
* SPI受験日
* インターンシップ
* 面接日程

#### 主な機能

* 月表示
* 日付選択による詳細表示

---

### 5.3 企業一覧画面

#### 表示内容

* 登録企業一覧
* 選考進捗状況

#### 主な機能

* 企業検索
* 企業情報編集
* 企業削除

---

### 5.4 企業追加画面

#### 入力項目

* 企業名
* ID
* パスワード
* URL
* メールアドレス
* ES提出日
* ES提出状況
* 顔写真提出日
* SPI提出日
* インターンシップ予定日
* 第1次面接日
* 第2次面接日
* 第3次面接日
* 備考

#### 主な機能

* 入力内容の保存
* 入力内容の編集

---

### 5.5 通知設定画面

#### 主な機能

* メール通知のON/OFF
* 通知タイミング設定

---

## 6. 非機能要件

* 直感的に操作できるUIを提供する。
* パソコンで快適に利用できるレスポンシブデザインとする。
* IDやパスワードなどの情報を安全に保存する。
* データの保存・更新を高速に行えるようにする。
