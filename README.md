# 就職活動支援アプリ 要件定義書

## 1. システム概要

本システムは、大学生の就職活動における企業管理や選考スケジュール管理を支援することを目的とし、企業ごとのエントリー情報や選考状況、提出期限などを一元管理し、一覧表示やカレンダー表示、通知する

---

## 2. システムの目的

* 就職活動に関する情報を一元管理する。
* ES提出や面接日程などの重要な期限を管理する。
* 通知機能により提出忘れや日程の見落としを防止する。
* 学生の就職活動の負担を軽減する。

---

## 3. 利用対象者

* 就職活動を行う大学生
* インターンシップや本選考を受ける学生

---

## 4. 機能要件

### 4.1 企業情報管理機能

企業ごとに選考状況や提出期限を登録・編集・削除できる。

#### 登録項目

| 項目名    | 内容            |
| ------ | ------------- |
| 企業名     | 応募企業名              |
| マイページURL      | 企業のマイページURL       |
| マイページID   | 企業サイトのログインID    |
| 備考      | 自由記述欄              |
| ES締切日  | エントリーシートの提出期限 |
| SPI提出日 | SPI受験期限       |
| インターンシップ日 | インターンシップの日程       |
| 第1次面接日 | 一次面接の日程       |
| 第2次面接日 | 二次面接の日程       |
| 第3次面接日 | 三次面接の日程       |
| 顔写真提出日     | 顔写真の提出期限         |

---

### 4.2 一覧表示機能

登録されたタスクやイベントを一覧形式で表示する。

#### 機能内容

* 期限が近い順に表示する。
* 未完了のタスクを優先して表示する。
* ES提出や面接予定を一覧で確認できる。
* 完了済みタスクの表示切替を行う。

---

### 4.3 カレンダー表示機能

登録された予定をカレンダー形式で表示する。

#### 機能内容

* ES提出日を表示する。
* SPI受験日を表示する。
* 面接日程を表示する。
* 日付を選択すると詳細情報を確認できる。

---

### 4.4 通知機能

登録された予定の期限が近づいた際に通知を行う。

#### 通知方法

* アプリ内通知
* メール通知

#### 通知設定

* 通知機能のON/OFF切替
* 通知タイミング設定
  * 当日
  * 1日前
  * 3日前
  * 1週間前

---

# アプリの使い方

## 文字入力について

日本語を入力する場合は、メモ帳などで文字を入力してコピーし、
入力欄を選択した状態で貼り付けてください。
英数字は入力欄へ直接入力できます。

## 1. 起動方法

1. Processingで `app` フォルダ内の `app.pde` を開く。
2. 実行ボタン（▶）を押す。
3. アプリ画面が表示される。

## 2. 画面の移動方法

画面下部のタブをクリックして、各画面へ移動できます。

- ToDoリスト：登録済みのタスクを確認する。
- カレンダー：予定をカレンダー形式で確認する
- 企業一覧：登録企業を確認・検索・編集・削除する
- 追加・編集：企業情報を新規登録する
- 通知設定：通知のON/OFFや通知タイミングを設定する

## 3. 企業情報を追加する

1. 「企業一覧」画面で「＋ 新規企業を追加」をクリックもしくは「追加・編集画面」をクリックする。
2. 企業名、締切日、面接日などを入力する。
3. 「保存する」をクリックする。
4. 企業一覧に登録内容が表示される。

### 日付の入力形式

日付は `2026/7/22` または `2026/07/22` の形式で入力できます。
未入力でも保存できますが、存在しない日付は入力できません。

## 4. カレンダーを確認する

1. 下部の「カレンダー」タブをクリックする。
2. `＜` と `＞` をクリックして、2026年7月から12月までの月を切り替える。
3. 予定がある日をクリックする。
4. 右側の「予定詳細」で企業名、予定内容、選考状況を確認する。

## 5. 企業情報を編集・削除する

- 編集：企業一覧の「編集」をクリックし、内容を変更して保存する。
- 削除：企業一覧の「削除」をクリックする。
ToDoリスト内の登録した画面をクリックしても追加・編集できる。

## 6. 通知を設定する

1. 下部の「通知設定」タブをクリックする。
2. アプリ内通知をONまたはOFFにする。
3. 通知タイミングを選択する。
